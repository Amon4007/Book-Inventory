# Book Inventory Management CLI

This is a simple Command-Line Interface (CLI) application for managing a book inventory. Users can add, view, update, and delete books stored in an SQLite database. By Amon Chirchir

## Features

- **Add a Book**: Add a new book with title, author, and status.
- **View All Books**: Display all books in the inventory.
- **Update Book Status**: Update a book's status to 'Available' or 'Checked Out'.
- **Delete a Book**: Remove a book from the inventory.
